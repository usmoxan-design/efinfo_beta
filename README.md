# 📱 eFinfo HUB

**eFinfo HUB** — bu barcha foydali ma'lumotlar, yangiliklar va raqamli resurslarni bir joyga jamlagan universal platforma. Foydalanuvchilar uchun qulay interfeys va tezkor ishlash imkoniyatini taqdim etadi.

-----

## 🚀 Xususiyatlari

  * **Markazlashgan Ma'lumotlar:** Barcha kerakli ma'lumotlar bir nuqtada.
  * **Tezkor Qidiruv:** Kerakli kontentni soniyalar ichida topish imkoniyati.
  * **Push-bildirishnomalar:** Eng so‘nggi yangilanishlardan doimo xabardor bo‘ling.
  * **Foydalanuvchi Profili:** Shaxsiy sozlamalar va saqlangan ma'lumotlar boshqaruvi.
  * **Tun/Kun Rejimi:** Ko‘zni charchatmaydigan qorong‘u rejim (Dark Mode) mavjud.

## 📦 O'rnatish (Installation)

Loyihani o'z qurilmangizda ishga tushirish uchun quyidagi qadamlarni bajaring:

1.  **Repozitoriyani klonlash:**
    ```bash
    git clone https://github.com/username/efinfo_beta.git
    ```
2.  **Loyiha papkasiga o'tish:**
    ```bash
    cd efinfo_beta
    ```
3.  **Bog'liqliklarni o'rnatish:**
    ```bash
    flutter pub get
    ```
4.  **Ilovani ishga tushirish:**
    ```bash
    yoki flutter run
    ```
